# ROS-ROS2-BOOKS
《从ROS1到ROS2无人机编程实战指南》

首先创建文件用于保存所有的代码

```
## 创建工作空间
$ mkdir -p workspace/src && cd workspace/src
## 将文件和submodule一起拉出
$ git clone --recursive https://github.com/lovelyyoshino/ROS-ROS2-BOOKS.git
```
